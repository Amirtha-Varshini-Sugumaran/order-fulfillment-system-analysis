# User Stories

## US-01: Create Order

As an operations coordinator, I want to create a new delivery order so that customer requests are captured in one place.

## US-02: Assign Delivery

As an operations coordinator, I want to assign a delivery to a driver so that the order can move from pending to planned.

## US-03: View Active Orders

As an operations manager, I want to view all active orders so that I can understand the current workload.

## US-04: Update Delivery Status

As a driver, I want to update the delivery status so that operations and customer support know what is happening.

## US-05: Search Customer Order

As a customer support agent, I want to search for an order so that I can answer customer questions quickly.

## US-06: Flag Delayed Delivery

As an operations manager, I want delayed deliveries to be highlighted so that the team can respond before the customer complains.

## US-07: View Driver Workload

As an operations coordinator, I want to see how many active deliveries each driver has so that I can assign work fairly.

## US-08: View Status History

As an operations manager, I want to see status history for an order so that I can understand what happened if there is a dispute.

## US-09: View Daily Dashboard

As a senior manager, I want to view daily delivery KPIs so that I can track service performance.

## US-10: Customer-Friendly Status

As a customer support agent, I want to see a customer-friendly delivery message so that I can give a clear update without reading internal notes.
